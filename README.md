## 🇬🇧 English

### About This Project
This repository contains my hands-on practice in Web Application Security using DVWA.

I focus on identifying and exploiting common vulnerabilities such as:
- SQL Injection
- Cross-Site Scripting (XSS)
- Authentication Bypass
- HTTP Request Manipulation (Burp Suite)

### Tools & Environment
- DVWA (Damn Vulnerable Web Application)
- Kali Linux
- Burp Suite

### Goal
To improve my practical cybersecurity skills and build a professional portfolio.
